# 圖片放置說明

請將以下圖片放在此資料夾中：

## 御守圖片
- **檔名**: `omamori.png` (或 `.jpg`)
- **用途**: 平安御守的正面圖片
- **建議尺寸**: 寬高比約 2:3，解析度至少 400x600px

## 資訊卡圖片
- **檔名**: `info-card.png` (或 `.jpg`)
- **用途**: 留守人資訊卡的背景圖片
- **建議尺寸**: 寬高比約 2:3，解析度至少 400x600px

## 角色圖片
請將四個角色圖片放在此資料夾：

1. **領隊 (Leader)**
   - **檔名**: `role-leader.png` (或 `.jpg`)
   - **建議尺寸**: 至少 200x200px

2. **廚師 (Chef)**
   - **檔名**: `role-chef.png` (或 `.jpg`)
   - **建議尺寸**: 至少 200x200px

3. **攝影師 (Photographer)**
   - **檔名**: `role-photographer.png` (或 `.jpg`)
   - **建議尺寸**: 至少 200x200px

4. **旅行者 (Traveler)**
   - **檔名**: `role-traveler.png` (或 `.jpg`)
   - **建議尺寸**: 至少 200x200px

## 注意事項

- 如果圖片不存在，系統會自動使用備用的 SVG 圖示
- 支援的格式：`.png`, `.jpg`, `.jpeg`, `.webp`
- 建議使用透明背景的 PNG 格式以獲得最佳效果
