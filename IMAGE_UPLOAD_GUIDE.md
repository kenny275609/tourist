# 圖片替換功能說明

## 功能概述

管理員現在可以替換首頁的海拔剖面圖。您可以**直接上傳圖片**或使用圖片 URL 來替換預設的 SVG 圖表。

## 使用步驟

### 方法 1：直接上傳圖片（推薦）✨

1. **設定 Supabase Storage**
   - 請先參考 [STORAGE_SETUP.md](./STORAGE_SETUP.md) 完成 Storage 設定
   - 創建名為 `trip-images` 的公開 bucket
   - 設定 Storage 政策

2. **上傳圖片**
   - 以管理員身份登入
   - 前往「設定」頁面（底部導航欄）
   - 在「行程設定」區塊中找到「海拔剖面圖圖片」
   - 點擊「上傳圖片」按鈕
   - 選擇本地圖片文件
   - 系統會自動上傳並顯示預覽
   - 上傳完成後會自動儲存

3. **清除圖片**
   - 點擊圖片預覽右上角的「X」按鈕
   - 或清空 URL 欄位
   - 點擊「儲存設定」恢復預設 SVG

### 方法 2：使用圖片 URL

1. **準備圖片**
   - 將圖片上傳到圖床服務（如 Imgur、Cloudinary、或您自己的伺服器）
   - 取得圖片的公開 URL

2. **設定圖片 URL**
   - 以管理員身份登入
   - 前往「設定」頁面（底部導航欄）
   - 在「行程設定」區塊中找到「海拔剖面圖圖片」欄位
   - 在 URL 輸入框中貼上圖片的 URL
   - 系統會自動顯示預覽
   - 點擊「儲存設定」

## 圖片建議

- **尺寸**：建議寬度 800-1200px，高度 300-500px
- **格式**：支援 JPG、PNG、WebP、GIF 等常見格式
- **大小限制**：上傳圖片限制為 5MB
- **內容**：可以是手繪風格的海拔剖面圖、路線圖、或其他相關圖片

## 注意事項

- **上傳功能**：需要先設定 Supabase Storage（參考 [STORAGE_SETUP.md](./STORAGE_SETUP.md)）
- **圖片 URL**：如果使用 URL 方式，圖片必須是公開可訪問的
- **自動回退**：如果圖片載入失敗，系統會自動回退到預設的 SVG 圖表
- **自動調整**：圖片會自動調整大小以適應容器
- **權限**：只有管理員可以修改圖片設定

## 故障排除

如果圖片無法顯示：

1. 檢查圖片 URL 是否正確
2. 確認圖片是公開可訪問的（不需要登入）
3. 檢查圖片格式是否支援
4. 如果問題持續，可以清除圖片 URL 恢復預設顯示
